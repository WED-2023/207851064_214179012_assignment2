About This Project

Submitted by: Naveh Gershoni (207851064), Harel Brener (214179012)

Templates used: No ready-made templates were used to build the site. All HTML, CSS, and JavaScript elements were written manually and customized, with the goal of deeply understanding the code structure, how to work with HTML elements, designing using CSS, and writing game logic using JavaScript.

jQuery plugins used: No jQuery plugins were used. All work was done using pure JavaScript, including switching between screens, handling the registration and login forms, running the game logic, and managing the shooting, enemies, scoring, and sounds.

Challenges we encountered:

  • Implementing the game logic – managing the player's movement, enemies, and shooting in a smooth (and not fragmented) manner, with identifying vulnerabilities and managing life and scoring, was a design and logical challenge.
  • Design and screen adaptation – We wanted to create pages that were easy to use and pleasing to the eye, so the use of CSS included contrasting colors, formatted tables, prominent buttons, and more.
  • Controlling navigation between screens – It was challenging to maintain proper navigation while maintaining access to only allowed screens (such as setting a game to only be played after login).
  • Adding sounds - understanding how to work with audio, from where to download it to where to upload it, and how to integrate it into the script at the exact location was a logical and technical challenge.
